# NASA/POWER Agrometeorological Dataset

This directory contains the historical meteorological data used in the Q-SARIMA experiments. 

* **Source:** NASA Prediction of Worldwide Energy Resources (POWER) API.
* **Format:** `.json` files.
* **Variables:** Temperature, Relative Humidity, Wind Speed, and Solar Radiation.
* **Climates:** Tropical Monsoon (Am) and Tropical Savanna (Aw) locations in Brazil.

**Note:** The data has been aggregated into decendial (10-day) periods during the preprocessing phase of the main script.
